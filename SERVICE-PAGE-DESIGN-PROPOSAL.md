# サービスページ デザイン提案書

## 全体コンセプト: 「捨てない遺品整理」
「捨てるのではなく、つなぐ。」をページ全体の軸とし、リユース・リサイクルの取り組みで差別化する。

## 設計の基本方針
- **ターゲット**: 50代~80代の遺族 + 代理検索する30代~50代の子世代
- **ユーザー心理5段階**: 感情的混乱→業者選定の不安→費用の懸念→環境配慮→行動への躊躇
- **配色**: 既存のグリーン系を踏襲（安心・誠実のイメージに最適）
- **モバイルファースト**: スマートフォンからの問い合わせが主流

## 掲載必須コンテンツ
1. 遺品整理サービスの説明
2. 3Rディレクター資格保有者在籍のアピール
3. JRRC（遺品3Rディレクター協会）の保険加入で1億円まで保障
4. 回収品のリユース・リサイクル（海外輸出含む）の取り組み

## ページ構成（13セクション）

### セクション1: ヒーローセクション（ファーストビュー）
- **心理段階**: 第1段階（感情的混乱）に対応
- **レイアウト**: フルワイド背景画像 + オーバーレイテキスト
- **コピー案**:
  - メイン: 「大切な方の想い出を、心を込めてお手伝いします」
  - サブ: 「埼玉県全域対応。はじめての遺品整理、不安なお気持ちに寄り添います」
- **CTA**: 「無料お見積り・ご相談」ボタン + 電話番号
- **3秒ルール対応**: 共感コピー + CTAのみに絞り、情報過多を避ける

### セクション2: 信頼バッジバー（★新規追加）
- **心理段階**: 第2段階（業者選定の不安）に即座に対応
- **レイアウト**: 横並びアイコン+テキスト（PC: 5列 / モバイル: 2x3グリッド）
- **内容**: 「遺品整理士」「3Rディレクター」「JRRC保険1億円」「年間実績XXX件」「埼玉県全域対応」
- **デザイン**: 薄いグリーン背景バー、シールド/チェックマークアイコン

### セクション3: サービス概要
- **心理段階**: 「何をしてくれるのか」を明確化
- **レイアウト**: 3カラムカード型（モバイル: 縦積み1カラム）
- **コピー案見出し**: 「遺品整理のすべてをお任せください」
- **カード内容**:
  - 遺品の仕分け・整理（遺族の意向を確認しながら丁寧に）
  - 買取・査定サービス（買取専門店運営の経験を活かし適正価格で）
  - リユース・リサイクル（3Rディレクターによる国内外での再利用）

### セクション4: 「捨てない片付け」セクション（★新規追加・最大の差別化ポイント）
- **心理段階**: 第4段階（環境・社会的配慮）に対応
- **レイアウト**: ジグザグ型（左テキスト/右写真を交互に）+ フローチャート
- **キャッチコピー**: 「捨てるのではなく、つなぐ。」
- **コピー案見出し**: 「大切なお品物を、次の方へつなぎます」
- **コンテンツ**:
  - 3Rディレクター資格の意義（JRRC公式認定、法令遵守・高い倫理性の証明）
  - 海外輸出を含むリユースの流れをフローチャートで図解（フィリピン、カンボジア等の国名例示）
  - 「捨てるのではなく、活かす」という情緒的価値の訴求
  - 和服・着物の海外リユース需要が特に高いことを強調
  - エシカル消費の文脈で若い世代にも訴求
  - 買取による実質費用削減メリットも併記

### セクション5: サービスの流れ（5ステップ）※既存を維持
- **レイアウト**: 縦型タイムライン（番号アイコン + 線でつなぐ）
- **ステップ**: お問い合わせ → 無料現地調査 → ご契約 → 作業実施 → 完了・精算

### セクション6: 料金の透明性（★新規追加）
- **心理段階**: 第3段階（費用への懸念）に対応
- **レイアウト**: 間取り別の料金目安テーブル + 注意書き
- **コンテンツ**:
  - 間取り別の参考料金（1K / 1LDK / 2LDK / 3LDK）
  - 「追加料金は一切いただきません」の明示
  - 「買取による費用削減」のメリット訴求
  - 「正確な料金は無料現地調査後にお見積り」

### セクション7: 中間CTA（★新規追加）
- **レイアウト**: フルワイドのグリーン濃いめ背景バー
- **コピー案**: 「まずはお気軽にご相談ください」「お見積りは完全無料。ご相談だけでも大歓迎です」
- **CTA（3チャネル）**:
  - 電話: 「今すぐ電話で無料相談する」
  - メール: 「メールで無料見積りを依頼する」（24時間受付）
  - LINE: 「写真を送るだけで簡単お見積り」（モバイルで特に目立たせる）
- **追加文言**: 「強引な営業は一切いたしません」

### セクション8: 安心の保障・資格セクション（★新規追加・信頼要素の深掘り）
- **レイアウト**: 3カラム横並びカード型（モバイル: 縦積み）
- **カード1 - 3Rディレクター**:
  - バッジ型ビジュアルアイコン
  - 「JRRC認定"遺品3Rディレクター"が対応します」
  - 消費者トラブル増加を背景に創設された専門資格
  - 法令順守の研修・試験を通過した専門資格であることを説明
- **カード2 - JRRC保険1億円**:
  - 「1億円」を大きなフォントで表示
  - 盾アイコンで「保護されている」イメージ
  - 保障対象の具体例: 作業中の建物・壁の破損、お客様の大切な品物の損傷、搬出時の事故
- **カード3 - プライバシー保護**:
  - 「個人情報・機密書類の適切な処理」
  - 機密文書のシュレッダー処理対応
  - 故人の手紙・写真など、デリケートな品物への配慮

### セクション9: 買取対象品目 ※既存を維持
- **レイアウト**: アイコングリッド（PC: 4x3 / モバイル: 3x3）
- **品目**: 大型家具、車・バイク、電化製品、民芸品、貴金属・宝石、ブランド品、着物（目立つ位置に）、書籍、楽器 等
- **補足**: 「他にも買取できるものがございます」

### セクション10: 対応エリア ※既存を維持
- **レイアウト**: 地図画像 + テキストリスト
- **モバイル**: 地図上、リスト下の縦積み

### セクション11: お客様の声（★新規追加）
- **レイアウト**: カード型スライダー/カルーセル（横スワイプ対応）
- **内容**: 3件以上の実例（年齢、状況、感想、星評価）

### セクション12: よくある質問（★新規追加）
- **レイアウト**: アコーディオン型（開閉式）
- **質問例**:
  - 「見積り後に追加料金はかかりますか？」→ 「一切かかりません」
  - 「作業当日に立ち会いは必要ですか？」
  - 「賃貸物件の原状回復も対応できますか？」
  - 「遺品の供養はお願いできますか？」
  - 「まだ遺品整理するか決めていないのですが相談だけでもいいですか？」→ 「もちろんです」
  - 「故人の個人情報や手紙はどう扱われますか？」→ 適切に処理、シュレッダー対応

### セクション13: 最終CTA ※既存CTAを強化
- **レイアウト**: フルワイド、グリーン系グラデーション背景
- **コピー案**: 「お見積り・ご相談は完全無料です」「強引な営業は一切いたしません」
- **CTA**: 電話（大きく表示）+ 見積りボタン + LINEボタン + メールリンク

## CTA配置戦略（計4箇所）

| 配置 | セクション | タイプ | 目的 |
|------|-----------|--------|------|
| 上部 | 1（ヒーロー） | ボタン+電話 | 即決層を逃さない |
| 中間 | 7（中間CTA） | フルワイド3チャネル | 情報取得後の行動促進 |
| 下部 | 13（最終CTA） | フルワイド複数チャネル | 全情報読了後の行動 |
| 常時 | フローティング | モバイル固定フッター | いつでもアクセス可能（既存維持） |

## 信頼要素の二段構え戦略
- **即座に（セクション2）**: バッジバーでコンパクトに5つの信頼要素を表示 → 第一印象で信頼感形成
- **詳しく（セクション4, 8）**: 3Rディレクターをストーリーとして差別化訴求 + 保険・資格の詳細をカード型で提示

## モバイル設計の要点
- タップターゲット: 全ボタン48px以上
- 電話番号: tel:リンクでワンタップ発信
- フローティングCTA: スクロール時に画面下部固定（既存維持）
- フォントサイズ: 本文16px以上、見出し24px以上
- カラム変更: PC 2~3カラム → モバイル 1カラム
- LINE CTA: モバイルで特に目立たせる

## トーン&マナー
- 既存サイトの「安心・明朗・丁寧」のトーンを継承
- 専門用語は噛み砕いて、やさしい日本語で伝える
- 感情的訴求（共感・安心）と論理的訴求（資格・保障・数字）をバランスよく配置
- 「無料」「気軽に」「ご相談だけでも」を繰り返し、心理的ハードルを下げる

## 既存デザインからの変更点まとめ
- グリーン系配色・トーン&マナー → **そのまま維持**
- 既存のサービスフロー・買取品目・対応エリア・CTA・フローティングボタン → **構造を維持**
- **新規追加**: 信頼バッジバー / 「捨てない片付け」セクション / 料金透明性 / 中間CTA / 資格・保障3カラム / お客様の声 / FAQ / プライバシー保護
